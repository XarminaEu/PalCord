Datenschutzerklärung

Letzte Aktualisierung: 22. Juni 2026

1. Verantwortlicher

PalCord wird betrieben von RL-Dev.de.

2. Welche Daten werden verarbeitet?

Bei der Nutzung von PalCord werden folgende Daten verarbeitet:

  • Discord-Profildaten (Discord-ID, Name, Avatar, Server-Mitgliedschaften) für Login und Dashboard-Zugriff
  • Palworld-Spielerdaten (UserId, Ingame-Name, Level, Spielzeit, Coins) aus dem Server-RCON/REST-API
  • Server-Konfigurationen (RCON-Daten, Channel-IDs, Rollen-IDs) für die Bot-Funktionen
  • Logs und technische Daten für Fehleranalyse und Copyright-Prüfung

3. Speicherung

Die Daten werden lokal in einer SQLite-Datenbank auf dem Server des Betreibers gespeichert. Es werden keine Daten an Dritte weitergegeben, außer an die konfigurierten Palworld-Server per RCON/REST-API.

4. Rechte

Benutzer können jederzeit die Löschung ihrer Daten beim Server-Betreiber beantragen.

5. Cookies

Das Dashboard verwendet Session-Cookies für den Login und ein Sprach-Cookie für die Spracheinstellung.
